
# Application Chancify

This is a chancify to write applications quickly. It contains a few
main features:

Core Features
- [x] Authentication
- [x] TRPC Api routes
- [x] DynamoDB Database
- [x] Data Models
Integrations
- [x] File Uploads with S3
- [ ] Message Queues
- [ ] Scheduled Jobs
- [ ] Emails
- [ ] Payments
- [ ] Text Messaging
- [ ] Middleware
Components
- [ ] Component Lists
- [x] Interactive Frames
- [x] Prose Writing

To get started just run:

```bash
  pnpm install
  pnpm run dev
```
